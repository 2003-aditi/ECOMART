# ECOMART
EcoMart leverages Excel for financial analysis, SQL for structured queries, Python for predictive modeling, and Power BI for interactive dashboards. It enables sales forecasting, inventory optimization, ROI evaluation, and KPI visualization, reducing stockouts, minimizing holding costs, and improving overall retail profitability.
Dataset Contents


## Overview
EcoMart is a retail analytics project designed to optimize inventory, forecast demand, and evaluate ROI. It integrates Excel, SQL, Python, and Power BI to deliver actionable insights for business decision‑making.

# Dataset Contents
Financial_Analysis: Holding costs, stock‑on‑hand, stockout counts.
ROI_Calculator: Stockout penalties, lost revenue, projected savings.
Sales_Data: Daily transactions (Jan–Apr 2025) with product ID, category, units sold, discounts, and revenue.

# Tools & Usage
Excel → Financial analysis, ROI tracking.
SQL → Structured queries, data extraction.
Python → Predictive modeling, feature engineering, AI‑driven inventory optimization.
Power BI → KPI dashboards, interactive visualizations.

# Use Cases
Sales forecasting
Inventory optimization
Profitability and ROI evaluation
Category‑level performance comparison
